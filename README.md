# changebook

Final project at the Ironhack Web Development Bootcamp.

Instructions to make it run:

clone the project
```
git clone https://github.com/uesteibar/changebook.git
```

install the ruby dependencies
```
bundle install
```

install the node dependencies
```
npm install
```

run the migrations
```
rake db:migrate
```

run the server
```
rails s
```


to run the specs:
````
rake spec
```
